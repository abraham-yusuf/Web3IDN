### Dimana letak halaman ?

Semua file mdx terletak di folder `/content`

| Dokumentasi          | Blog        | Learn         |
| ------------- | ------------- | ------------- |
| `/content/docs` | `/content/blog` | `/content/learn` |

### Pembaruan metode impor gambar

Semua gambar harus ditempatkan di folder `/public` dan direferensikan langsung:

```mdx
![Foto ku](/my-photo.png)
```

Untuk gambar dalam subdirektori, gunakan path lengkap dari folder public:

```mdx
![alt](/docs/my-photo.png)
![alt](/blog/my-photo.png)
![alt](/learn/my-photo.png)
```

## Bagaimana cara menambahkan halaman baru

<div className="steps-container">
  
### Buat file .mdx

Harap isi semua bidang

#### Untuk Dokumentasi (Docs)

Untuk membuat halaman baru di dokumentasi, buatlah file di folder `/content/docs`

```mdx
---
title: Judul disini
description: Deskripsi disini
---

# Judul / Heading 1

Teks disini...
```

#### Untuk Blog

Untuk membuat postingan baru di blog, buatlah file di folder `/content/blog`

```mdx
---
title: Judul Postingan
description: Deskripsi singkat blog?
authors: [<nama-authors>]
date: 2023/1/1
image: "/blog/banner.jpg"
tags: ["buat-tags"]
---
```

**Buatlah profile authors Anda sendiri**

Buka file `config.ts`, gulir dan cari baris ke `blogAuthors` dan tambahkan informasi pribadi Anda

- Hindari teks yang panjang
- Jangan memposting URL yang mencurigakan

```ts
export const blogAuthors: Record<string, AuthorData> = {
  //...
  abraham: {
    name: "Abraham Yusuf",
    title: "Bio anda",
    url: "https://www.abrahamyusuf.my.id",
    image_url: "/blog/abraham-yusuf.png", //masukkan gambar ke /public/blog/abraham-yusuf.png
  },
};
```

Anda kemudian dapat menggunakan `abraham` sebagai ID authors/penulis Anda

#### Untuk Learning Path

Untuk membuat kursus baru di Learning path, buatlah file di folder `/content/learn`

```mdx
---
title: Judul pelajaran
description: Deskripsi singkat pelajaran?
authors: [<nama-authors>]
date: 2023/1/1
image: "/learn/banner.jpg"
tags: ["buat-tags"]
---
```

**Buatlah profile authors Anda sendiri**

Buka file `config.ts`, gulir dan cari baris ke `learnAuthors` dan tambahkan informasi pribadi Anda

- Hindari teks yang panjang
- Jangan memposting URL yang mencurigakan

```ts
export const learnAuthors: Record<string, AuthorLearnData> = {
  //...
  abraham: {
    name: "Abraham Yusuf",
    title: "Bio anda",
    url: "https://www.abrahamyusuf.my.id",
    image_url: "/learn/abraham-yusuf.png", //masukkan gambar ke /public/blog/abraham-yusuf.png
  },
};
```

Anda kemudian dapat menggunakan `abraham` sebagai ID authors/penulis Anda

### Membuat Konten

Saya yakin Anda telah mempelajari beberapa sintaks Markdown dasar

#### Perhatikan hal berikut

- kata yang sesuai
- Tidak ada promosi tentang personal
- Tidak boleh bias atau politis

#### Tingkatkan artikel

- **Tipografi yang baik:**
  Gunakan judul sekiranya banyak teks yang berantakan
- **Spasi antara angka dan bahasa:**
  Semuanya mengikuti [Panduan Penulisan Copywriting Bahasa Indonesia](https://glints.com/id/lowongan/copywriting-adalah/#.ZEuwaN3P3b0)
- **Menggunakan komponen kostum:**
  Anda dapat mempelajari lebih lanjut [dari sini](./components)

### Mengurutkan halaman

Anda harus mengedit `meta.json` untuk mengubah urutan setiap halaman:

```json
{
  "my-new-page": "judul halaman baru",
  "index": "mulai cepat",
  "commands": "petunjuk"
}
```
